# Stock Price Prediction Using RNN

## Objective
To predict stock prices using Recurrent Neural Networks (RNN) and historical stock market data.

## Tools Used
- Python
- Google Colab
- TensorFlow
- Keras
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Historical stock market dataset containing:
- Stock 1
- Stock 2
- Stock 3

## Methodology
1. Data preprocessing
2. Feature scaling using MinMaxScaler
3. Sequence generation
4. RNN model construction
5. Training and validation
6. Prediction
7. Performance evaluation

## Results
The model successfully predicts stock price trends using historical patterns.
